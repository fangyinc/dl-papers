## 沐神的读论文方法

### 清楚论文结构

一般如下：

1. title
2. abstract
3. intro(导言：介绍)
4. 相关工作（本文提出的算法(method))
5. 试验exp
6. conclusion

### 三遍读论文

#### Pass1

1. 先读摘要，然后直接跳结论（通常与摘要类似，但是添加实际的结论和数字加以证明）
2. 跳到实验中和方法中，看看图和表。

pass1确定是否继续往下读

#### Pass2

1. 从标题一直往下读到最后，不用搞懂每个细节，但需要搞定每个实例的图和表，作者的方法和别人方法的对比。
2. 可以圈出相关文献

如果当前文献读不懂可以先读引用的先关文献。

#### Pass3

最后一遍，需要知道每一句话在干什么。
